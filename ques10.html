Difference between HTML and HTML5

HTML HTML5
Relies on third-party plugins (e.g., Flash) for multimedia content Native support for multimedia elements with <audio> and <video> tags
Limited set of semantic tags Introduces new semantic tags like <header>, <nav>, <section>, <article>, <footer> , etc.
Basic form controls and limited validation options Introduces new input types (date, email, url, etc.) and form validation attributes (required, pattern, etc.)
Lacks built-in support for offline storage and application caching Provides offline storage with localStorage and sessionStorage, and offline application caching
Older versions may have inconsistent support across Improved browser compatibility and support for modern features and APIs
